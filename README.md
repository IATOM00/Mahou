# Mahou

Mahou is a Windows layout switcher built with WinForms and .NET Framework 4.5.

This working copy is prepared as a local, self-contained baseline for further modification:

- no runtime update checker;
- no auto-download or self-replacement script;
- no startup background network check;
- no update/proxy UI;
- local settings stay beside the executable in `Mahou.ini`;
- snippets stay beside the executable in `snippets.txt`.

Build from `Mahou.sln` or run `buildMahou.cmd` on a machine with the .NET Framework MSBuild toolchain available.
